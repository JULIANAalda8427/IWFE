Sobre Nosotros

Brindamos información sobre empleos en las empresas ubicadas en La Vega Cundinamarca, para facilitar a las personas encontrar una vacante de trabajo, así mismo las empresas podrán realizar una contratación más rápida y efectiva, de tal manera tener un impacto positivo, disminuyendo el desempleo.

MISIÓN

Brindar una mayor accesibilidad a trabajar para las personas en busca de emmpleo mediante la página web de igual modo aumentar la productividad de las empresas.

VISIÓN

Poder llevar nuestra página web a más lugares del país para tener un mayor impacto y así mismo crear una aplicación.

OBJETIVOS

Lograr disminuir el desempleo con el fin de mejorar la productividad, de modo que se facilitará a las personas encontrar un empleo mediante la página web.
